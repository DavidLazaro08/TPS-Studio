# TPS Studio

TPS Studio es una aplicación de escritorio desarrollada en Java, orientada al diseño y preimpresión de tarjetas plásticas CR80 y acreditaciones.

## Objetivo
Permitir la creación de plantillas con medidas reales, campos variables y exportación a PDF listo para impresión.

## Tecnologías
- Java
- JavaFX
- Arquitectura MVVM
- Base de datos local (pendiente de definir)

## Estado del proyecto
Fase inicial de desarrollo (estructura base y arranque).
